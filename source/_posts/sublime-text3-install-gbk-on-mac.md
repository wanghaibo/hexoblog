##安装package controller##
<https://sublime.wbond.net/installation>

##安装Codecs33##
* command+shift+p
* 键入package
* 选中install package
* 键入Codecs33
* 选中安装
* 重启sublime

##安装ConvertToUTF8##
* command+shift+p
* 键入package
* 选中install package
* 键入ConverToUTF8
* 选中安装
